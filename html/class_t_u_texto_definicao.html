bool TUTextoDefinicao::Run ( )

Roda vários casos de testes para verificar se está tudo funcionando como o esperado.

São passados várias definições inválidas e verificado se as exceções estão sendo lançadas como previsto. É passado também uma definição válida e feito o mesmo teste, verificando se o que foi Setado é igual ao argumento.

Retorna
true se as exceções foram capturadas corretamente e a definição é válida.
false se alguma exceção não foi devidamente capturada.
