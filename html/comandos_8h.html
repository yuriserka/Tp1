Descrição detalhada

Classes de Comandos para executarem certas ações.

Autor
Yuri Serka e Thiago Matos
Data
13 de Abril de 2018

Neste arquivo estão as classes de comando que terão inteligencia suficiente para executar certas ações e deixar o código das controladoras mais legível.

As classes de comando são usadas para aproximar a implementação ao contexto de interfaces gráficas.
