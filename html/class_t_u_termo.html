bool TUTermo::Run ( )

Roda vários casos de testes para verificar se está tudo funcionando como o esperado.

É passado um Termo válido e verificado se as exceções estão sendo lançadas como previsto e verificando se o que foi Setado é igual ao argumento.

Retorna
true se as exceções foram capturadas corretamente e o Termo é válido.
false se alguma exceção não foi devidamente capturada.
