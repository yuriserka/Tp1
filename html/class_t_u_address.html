bool TUAddress::Run ( )

Roda vários casos de testes para verificar se está tudo funcionando como o esperado.

São passados vários endereços inválidos e verificado se as exceções estão sendo lançadas como previsto. É passado também um endereço válido e feito o mesmo teste, verificando se o que foi Setado é igual ao argumento.

Retorna
true se as exceções foram capturadas corretamente e o endereço é válido.
false se alguma exceção não foi devidamente capturada.
